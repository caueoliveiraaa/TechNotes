
python -m venv <env_name>  -> creates a virtual environment
<env_name>\Scripts\activate.bat -> activate the virtual environment

echo This is some content > myfile.txt -> creates a new file "myfile.txt" with the text "This is some content"
the > symbol is used as a redirector to send the output of a command to a file

notepad myfile.txt -> opens a Notepad with a new or existing file, depending on whether myfile.txt already exists

cd/  ->  returns to parent directory.

type <some_file> -> displays the content of file

dir <folder_name> \s \a -> finds folders and returns paths according to the name <my_folder>
/s searches for the folder recursively in all subdirectories
/a displays only the path, without any additional information

tree [drive:][path] [/f] [/a] -> Graphically displays the folder structure of a drive or path.
/f -> Display the names of the files in each folder.
/a -> Use ASCII instead of extended characters.

more <file_name> -> is used to display the contents of a text file one screen at a time

rd /s /q <folder_name> -> deletes a folder that is not empty on cmd
/s (including all subdirectories) 
/q (quiet mode)
